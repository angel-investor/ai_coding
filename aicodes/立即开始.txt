============================================
    立即开始使用系统
============================================

⚠️ 重要提示
============================================

语音问答功能需要配置 DeepSeek API Key
否则会报错: "文本生成失败"


🚀 快速开始 (3 步)
============================================

第 1 步: 配置 API Key
--------------------------------------------

双击运行: 配置API_KEY.bat

这将自动创建 .env 文件
并配置好 DeepSeek API Key


第 2 步: 启动服务器
--------------------------------------------

双击运行: start_predict_server.bat

等待看到:
"Flask 服务器启动成功"


第 3 步: 访问系统
--------------------------------------------

打开浏览器访问:
http://localhost:5000/

点击功能卡片进入各模块


✅ 完整流程 (推荐)
============================================

1. 安装依赖
   install_dependencies.bat

2. 配置 API Key ⭐ 重要
   配置API_KEY.bat

3. 训练模型
   train_model_xgb.bat

4. 生成报告 (可选)
   generate_report.bat

5. 启动服务
   start_predict_server.bat

6. 访问系统
   http://localhost:5000/


📋 使用主菜单 (推荐)
============================================

双击运行: START.bat

菜单选项:
  1 - 安装依赖
  2 - 配置 API Key ⭐ 重要
  3 - 训练模型
  4 - 启动服务
  5 - 生成报告
  6 - 启动语音问答
  7 - 测试语音问答
  8 - 检查环境
  0 - 退出


🎯 三大功能模块
============================================

1. 疾病风险预测 🔬
   • 输入 11 项健康指标
   • 实时评估疾病风险
   • 访问: http://localhost:5000/web/predict.html

2. 数据分析报告 📊
   • 5 种交互式图表
   • 统计信息汇总
   • 访问: http://localhost:5000/analysis/report.html
   • 需要先运行: generate_report.bat

3. AI 语音问答 🎙️
   • 智能健康咨询
   • 语音合成播放
   • 访问: http://localhost:5000/web/qa_audio.html
   • 需要配置 API Key ⭐


⚠️ 常见错误及解决
============================================

错误 1: "文本生成失败"
--------------------------------------------
原因: 未配置 DeepSeek API Key

解决:
1. 运行: 配置API_KEY.bat
2. 重启服务器


错误 2: "模型未加载"
--------------------------------------------
原因: 未训练模型

解决:
1. 运行: train_model_xgb.bat
2. 重启服务器


错误 3: 数据分析报告 404
--------------------------------------------
原因: 未生成报告

解决:
1. 运行: generate_report.bat
2. 访问: http://localhost:5000/analysis/report.html
